Introduction to Python

Introduction
What is Python?

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991. It is used for:

  • web development (server-side),
  • software development,
  • mathematics,
  • system scripting.
What can Python do?
  • Python can be used on a server to create web applications.
  • Python can be used alongside software to create workflows.
  • Python can connect to database systems. It can also read and modify files.
  • Python can be used to handle big data and perform complex mathematics.
  • Python can be used for rapid prototyping, or for production-ready software development.
Why Python?

Python works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc). Python has a simple syntax similar to the English language. Python has syntax that allows developers to write programs with fewer lines than some other programming languages. Python runs on an interpreter system, meaning that code can be executed as soon as it is written. This means that prototyping can be very quick. Python can be treated in a procedural way, an object-oriented way or a functional way.

Getting Started
Running Python

Python is an interpreted programming language, this means that as a developer you write Python (.py) files in a text editor and then put those files into the python interpreter to be executed. The way to run a python file is like this on the command line (in Windows): C:\Users\Your Name>python helloworld.py

Command Line

To test a short amount of code in python sometimes it is quickest and easiest not to write the code in a file. This is made possible because Python can be run as a command line itself.

C:\Users\Your Name>python
Python 3.6.4 (v3.6.4:d48eceb, Dec 19 2017, 06:04:45) [MSC v.1900 32 bit (Intel)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>> print("Hello, World!")
Hello, World!

Your First Program - Hello World

In order to write a Python program file you will need a text editor such as VS Code. Within the text editor, type the below line and then save as a file such as helloworld.py

print("Hello, World!")

Note that by convention a python file is prefixed with .py to indicate that the contents are understandable by python. Once the file is created, run it as below

C:\Users\Your Name>python helloworld.py

Syntax
Spaces

Where in other programming languages the indentation in code is for readability only, the indentation in Python is very important as Python uses this to indicate a block of code. The number of spaces is up to you as a programmer, the most common use is four, but it has to be at least one. However:

  • Skipping indentations will give an error
  • Indentations have to be consitently the same throughout the block of code.

Allowed

if 5 > 2:
print("5 is great")

Not Allowed

if 5 > 2:
print("5 is great")

if 5 > 2:
print("5 is great")
print("5 is great")

Variables

Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.

Variable Names

A variable can have a short name (like x and y) or a more descriptive name (age, carname, total_volume). Rules for Python variables:

  • A variable name must start with a letter or the underscore character
  • A variable name cannot start with a number
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
  • Variable names are case-sensitive (age, Age and AGE are three different variables)
  • A variable name cannot be any of the Python keywords.

Variable Data Types

  • Text Type: str
  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool
  • Binary Types: bytes, bytearray, memoryview
  • None Type: NoneType

it is possible to specifically set the data type of a variable. For example: x = str("Hello World")

Conditions

Python supports the usual logical conditions from mathematics:

  • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than or equal to: a >= b

Python also supports if statements:

if 5 > 2:
print("5 is great")
elif 5 < 2:
print("5 is less")

Loops

The while Loop

With the while loop we can execute a set of statements as long as a condition is true.

i = 1
while i < 6:
print(i)
if i == 3:
break
i += 1

Note the use of the break keyword to exit the loop early.

The For Loop

Python also supports for loops.

fruits = ["apple", "banana", "cherry"]
for x in fruits:
print(x)

You can also use the range keyword in for loops.

for x in range(6):
print(x)

functions

Functions are defined by the 'def' keyword with spaces used to define the scope of the function. Arguments can be optionally defined. To call the function, type the function name with brackets containing any arguments. The code below illustrates these points.

def my_function(fname):
print("Hello " + fname)

my_function("Emil")
my_function("Fred")
my_function("Bill")

Object Oriented Programming

Almost everything in Python is an object, with its properties and methods. A Class is like an object constructor, or a "blueprint" for creating objects.

class MyClass:
x = 5

Now we can use the class named MyClass to create objects:

p1 = MyClass()
print(p1.x)

Init Method

All classes have a function called __init__(), which is always executed when the class is being initiated. Use the __init__() function to assign values to object properties, or other operations that are necessary to do when the object is being created:

class Car:
def __init__(self, model, type):
self.model = model
self.type = type

p1 = Car("Ford", "Saloon")
print(p1.model
print(p1.type)

Object Methods

Objects can also contain methods. Methods in objects are functions that belong to the object. Looking at the Car example, a print function can be added.

class Car:
def __init__(self, model, type):
self.model = model
self.type = type

def printMe(self)
print("This car is a " + p1.model + " " + p1.type)

p1 = Car("Ford", "Saloon")
p1.printMe()

Self Parameter

The self parameter is a reference to the current instance of the class, and is used to access variables that belong to the class. It does not have to be named self, you can call it whatever you like (however by convention it is), but it has to be the first parameter of any function in the class.

Pass Parameter

Class definitons cannot be empty, but if you for some reason have a class definition with no content, put in the pass statement to avoid errors.

Inheritance

Inheritance allows us to define a class that inherits all the methods and properties from another class. Parent class is the class being inherited from, also called base class. Child class is the class that inherits from another class, also called derived class.

Parent Class

Any class is a Parent class. You have been using this in the examples so far.

class Car:
def __init__(self, model, type):
self.model = model
self.type = type

def printMe(self)
print("This car is a " + p1.model + " " + p1.type)

# Create an instance of 'Car' and use its printMe function

p1 = Car("Ford", "Saloon")
p1.printMe()

Child Class

To create a class that inherits the functionality from another class, send the parent class as a parameter when creating the child class:

class Van(Car):
pass

This declares a 'Van' to be a child calss of 'Car'. The 'pass' is used as a van has no differing functionality to that of a 'Car' (as it is defined so far).

Super Function

Python also has a super() function that will make the child class inherit all the methods and properties from its parent.

class Van(Car):
def __init__(self, model, type):
super().__init__(model, type)

Methods and Properties

You can also add new methods and properties to the child object

class Van(Car):
def __init__(self, model, type):
super().__init__(model, type)
self.doors = 4

def describe(self):
print("Van has " + self.doors + " on " + self.type

Libraries

Consider a module to be the same as a code library. A file containing a set of functions you want to include in your application. There are a whealth of third party modules and built in modules that can be used and these help make Python such a rich and adaptable language. Usage:

import mymodule as mx

a = mx.doThisThing()

Some of the more common modules to draw upon:

  • NumPy is short for "Numerical Python".
  • Pandas is used to analyze data.
  • SciPy is a scientific computation library that uses NumPy underneath.
  • Django makes it easier to build web pages using Python.
  • PyGame is used for game development
  • TensorFlow is used for machine learning
  • Matplotlib is a 2D graph plotting library.
  • BeautifulSoup library is used for web scraping.

Each module is a topic in itelf and beyond the scope of this document. The above list is by no means an exhaustive list of the available and is more to be taken as illustrative of the breadth of topics that Python can be used in.
